# Explicació del Chatbot

## Descripció
Aquest repositori tracta sobre un **chatbot de futbol** creat per interactuar amb els usuaris i respondre preguntes relacionades amb aquest esport, com ara equips, jugadors o competicions.

## Objectiu
L’objectiu principal del projecte és **oferir informació de manera interactiva i senzilla** als usuaris interessats en el món del futbol.

## Tecnologies
El chatbot està **programat en Python**, un llenguatge adequat per al desenvolupament d’aquest tipus d’aplicacions.

## Contingut del repositori
El repositori conté el codi necessari per entendre el **funcionament bàsic del chatbot** i permet ampliar el projecte amb noves funcionalitats en el futur.

## Possibles millores
- Afegir més preguntes i respostes  
- Connectar el chatbot a APIs de futbol  
- Integrar-lo en una aplicació web
