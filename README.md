# explicació chatbot
Aquest repositori tracta sobre un chatbot de futbol creat per interactuar amb els usuaris i respondre preguntes relacionades amb aquest esport, com ara equips, jugadors o competicions. L’objectiu principal és oferir informació de manera interactiva i senzilla.

El chatbot està programat en Python, el repositori conté el codi necessari per entendre el funcionament bàsic del projecte i permet ampliar-lo amb noves funcionalitats en el futur.
